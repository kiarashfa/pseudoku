# Pseudoku

**Pseudo + Sudoku**  is a functional Sudoku generator and solver (with four difficulty levels inspired by the `Four Tempers`), that can import puzzles manually or from images, track your statistics, and experience a workplace-approved number-sorting environment.

This is a fan tribute to Apple TV+ Severance (2022–present), recreating Lumon's iconic terminal experience, allowing users to sort numbers, refine files, and work within a retro-futuristic MDR workstation.

Please enjoy each puzzle equally!
